# @strata/agent

A Claude Agent SDK agent that works a codebase through Strata's structural tools only — it has no filesystem tools.

Part of [Strata](https://github.com/ToddHebebrand/strata) — an agent-native structural code substrate: a queryable, transactional code graph that replaces files as an AI agent's interface to a codebase. See the repo for the architecture, the measured results, and the write-up. Most users want [`@strata/cli`](https://www.npmjs.com/package/@strata/cli).
