# @strata-code/render

Strata node graph → canonical TypeScript text, with a source map back to node IDs.

Part of [Strata](https://github.com/ToddHebebrand/strata) — an agent-native structural code substrate: a queryable, transactional code graph that replaces files as an AI agent's interface to a codebase. See the repo for the architecture, the measured results, and the write-up. Most users want [`@strata-code/cli`](https://www.npmjs.com/package/@strata-code/cli).
